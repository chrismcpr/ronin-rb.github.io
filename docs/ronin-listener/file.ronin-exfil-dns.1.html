ronin-exfil-dns 1 "2023-03-24" Ronin Exfil "User Manuals"

SYNOPSIS

ronin-exfil dns [options] DOMAIN

DESCRIPTION

Starts a DNS server for receiving exfiltrated data.

ARGUMENTS

DOMAIN The domain to receive DNS queries for.

OPTIONS

-H, --host IP The interface to listen on. Defaults to 0.0.0.0 if not given.

-p, --port PORT The port to listen on. Defaults to 53 if not given.

-h, --help Prints help information.
